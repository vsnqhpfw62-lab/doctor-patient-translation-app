Healthcare Doctor–Patient Translation Web App

This repository contains the frontend structure and backend prototype for a doctor–patient translation application built as part of Nao Medical’s take-home assignment.

The fully interactive and deployed version of this application is available here:

Live Demo on CodePen

Features

  • Doctor and Patient role-based chat
  • Language translation for messages
  • Audio recording and playback
  • Conversation logging
  • AI-powered conversation summary

Note: Due to time and environment constraints, the deployed version focuses on frontend functionality. Backend logic is included separately to demonstrate full-stack design intent.
